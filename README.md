# 耕地智能分析系统

## 项目简介

这是一个基于深度学习的农业遥感图像分析系统，用于：
- 识别卫星/航空影像中的耕地区域
- 对比不同年份的耕地变化
- 计算耕地面积（单位：亩）
- 生成可视化对比图

## 技术特点

- **AI引擎**：基于U-Net深度学习模型的图像分割
- **智能增量预测**：利用去年数据实现38.9倍加速
- **高精度识别**：支持亚米级分辨率（≤0.5米）
- **坐标系支持**：自动处理CGCS2000坐标系转换
- **可视化对比**：左右对比展示两年变化

## 文件说明

### 核心文件
- `耕地分析工具_图形界面.py` - 主程序（推荐使用）
- `耕地分析系统.py` - 分析引擎
- `耕地识别模型.h5` - 训练好的模型权重
- `耕地识别模型_基准数据.pkl` - 去年耕地基准地图

### 辅助文件
- `耕地识别模型训练(16).py` - 模型训练脚本
- `高精度颜色识别.py` - 辅助识别模块
- `耕地变化评价指标.py` - 专业评估工具

### 配置文件
- `校正配置.json` - 面积校正配置（自动生成）

## 使用方法

### 快速开始
```bash
python 耕地分析工具_图形界面.py
```

### 操作步骤
1. 选择"去年"TIF图像（或使用基准数据）
2. 选择"今年"TIF图像
3. 点击"开始分析"
4. 查看分析结果和可视化图

### 图像要求
- 格式：GeoTIFF（带地理坐标）
- 分辨率：建议≤0.5米/像素
- 坐标系：CGCS2000（会自动转换）

## 注意事项

1. **首次运行**：确保模型文件(.h5)和基准数据(.pkl)存在
2. **坐标系统**：自动检测并转换不同的CGCS2000分带
3. **内存需求**：大图像需要8GB以上内存
4. **GPU支持**：自动检测并使用GPU加速

## 输出结果

- 面积统计（亩）
- 变化分析
- 可视化对比图（左：去年，右：今年）
- 详细的变化分解（新增/减少耕地）

## 技术支持

- 基于TensorFlow 2.x
- 使用OpenCV进行图像处理
- 基于rasterio处理地理数据
- 支持Windows/Linux/macOS
